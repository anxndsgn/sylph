# 关于

这个项目是一个基于Next.js、Tailwind CSS和ShadcnUI构建的学术主页模板。设计简约现代，为学术作品提供了干净和专业的外观。它支持部署到Zeabur，并可以直接解析.bib文件以显示出版物。

<button className="bg-black-a1 dark:bg-white-a1 px-4 py-2 mt-4 rounded-md"> <a href='https://academic.anxn.me' target="_blank">在线预览</a> </button>

# 特性
- 简约设计：干净现代的界面，展示学术作品。
- Markdown支持：使用markdown文件编写网站内容。
- 暗黑模式：支持暗黑模式，提供舒适的查看体验。
- 响应式设计：完全响应式和移动友好。
- Bib文件解析：直接解析并显示.bib文件中的出版物。


更多信息，请参阅[Github仓库](https://github.com/anxndsgn/academic-homepage-template)。
